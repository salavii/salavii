## About Me

I am a Computer Science Master's student with a strong focus on artificial intelligence and machine learning. My primary interest lies in understanding and building large language models from first principles, with hands-on experience in transformer architectures, fine-tuning pipelines, and applied deep learning.

I believe in learning by building — every project in this portfolio reflects a deliberate effort to understand the underlying mechanisms, not just apply existing tools.

## 🚀 Featured Projects

### 🧠 [LLM From Scratch](https://github.com/salavii/llm-from-scratch)

A full implementation of a GPT-style language model built from first principles using PyTorch.
Covers tokenization, self-attention, transformer blocks, pretraining, classification fine-tuning, and instruction fine-tuning with GPT-2 weight loading.

`Python` `PyTorch` `tiktoken` `Jupyter`

---

### 🤖 [SOP Generator — Fine-tuning](https://github.com/salavii/SOP-Generator-Fine-tuning)

An LLM fine-tuning project focused on generating graduate school Statements of Purpose using instruction-tuning techniques.

`Python` `PyTorch` `NLP` `Fine-tuning`

---

### 🌿 [Herbarium Classification](https://github.com/salavii/Herbarium-Classification)

A CNN-based deep learning pipeline for classifying digitized herbarium plant specimens.
Includes data preprocessing, augmentation, training, and evaluation workflows.

`Python` `PyTorch` `CNN` `Computer Vision`

---

### 🌍 [EuroSAT Land Cover Classification](https://github.com/salavii/EuroSAT-Land-Cover-Classification)

Land cover classification on satellite imagery using deep learning.

`Python` `TensorFlow` `CNN` `Remote Sensing`
